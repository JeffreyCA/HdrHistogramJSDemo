# HdrHistogramJSDemo

Demo repository for HdrHistogramJS with support for memtier_benchmark output files.

## Prerequisites

- [Node.js](https://nodejs.org/)

## Setup

1. Clone the repository
2. Install the dependencies with `npm install`

## Local development

```sh
npm run dev
```

## Production build

```sh
npm run build
```

## Deploy to GitHub Pages

```sh
npm run deploy
```
